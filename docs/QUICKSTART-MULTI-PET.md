# 快速开始：多物种数据演示

## 🎯 5分钟快速体验

### 步骤1：导入多物种数据

```bash
cd pet-health-daily
node scripts/importMultiPetData.js
```

**等待30秒**，看到：
```
✅ 数据导入完成！
总计导入: 1680 条数据 (10 个宠物 × 7 天 × 24 小时)
```

### 步骤2：启动服务

```bash
# 终端1
npm run dev

# 终端2（新终端）
cd client
npm start
```

### 步骤3：打开浏览器

访问：http://localhost:3000

### 步骤4：切换宠物

1. 点击页面上方的宠物选择下拉框
2. 选择不同的宠物（如：豆豆、喵喵、大黄等）
3. 观察数据变化：
   - 头部显示当前宠物名称和图标
   - 步数、活动量差异
   - GPS轨迹位置不同
   - 体温等指标变化

## 💡 体验要点

### 1. 观察物种差异

**狗（5只）**：
- 步数：7000-12000步
- 特点：活动量大，GPS轨迹范围广

**猫（5只）**：
- 步数：3000-4500步
- 特点：活动量小，GPS轨迹范围小

### 2. 对比活动等级

- **极高**：大黄（12000步）
- **高**：豆豆、旺财、黑子（8000-10000步）
- **中**：喵喵、橘子、雪糕、芝麻（4000-7000步）
- **低**：咪咪、小白（3000-3500步）

### 3. 查看地域差异

每个宠物在不同城市：
- 豆豆：北京天安门附近
- 喵喵：上海外滩区域
- 旺财：深圳湾公园
- ...

## 📱 界面功能

### 头部
- 动态显示：`{宠物图标} {宠物名}的日报`
- 副标题：`{物种类型} • {宠物ID}`

### 选择器区域
1. **下拉菜单**：列出10个宠物
2. **信息卡片**（紫色）：
   - 大号宠物图标（跳动动画）
   - 宠物名称
   - 物种ID

### 日报数据
- 活动数据：步数、达标率
- 体征数据：体温、气压
- 趋势对比：今日 vs 昨日
- GPS地图：活动轨迹

## 🔍 API测试

```bash
# 豆豆的日报
curl http://localhost:3001/api/report/DOG001

# 喵喵的日报
curl http://localhost:3001/api/report/CAT001

# 大黄的轨迹（最活跃）
curl "http://localhost:3001/api/location/track/DOG003?date=$(date +%Y-%m-%d)"
```

## ⚡ 常见操作

### 切换宠物
1. 点击下拉框
2. 选择目标宠物
3. 自动加载数据

### 切换日期
1. 点击日期选择器
2. 选择日期（最近7天）
3. 查看历史数据

### 查看GPS轨迹
1. 滚动到地图模块
2. 查看轨迹线和标记点
3. 点击轨迹点查看详情
