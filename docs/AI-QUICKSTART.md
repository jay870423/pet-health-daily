# AI 深度健康洞察功能 - 快速开始

## 🎯 功能概述

新增的 AI 深度健康洞察板块使用 DeepSeek AI 对宠物健康数据进行专业分析，提供个性化建议。

## 🚀 快速配置（3步）

### 1️⃣ 获取 DeepSeek API Key

访问 [DeepSeek 平台](https://platform.deepseek.com/)：
- 注册账号
- 创建 API Key
- 复制 Key

### 2️⃣ 配置环境变量

编辑 `.env` 文件：
```env
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxx  # 替换为你的真实 API Key
```

### 3️⃣ 重启服务

```bash
# 停止当前服务（Ctrl+C）
# 重新启动
npm run dev
```

## ✅ 验证配置

1. 访问 http://localhost:3000
2. 在 AI 深度健康洞察卡片中点击"🚀 开始分析"
3. 看到实时分析过程即表示配置成功

## 🔧 故障排查

### 问题：显示"规则分析"而非"开始分析"
**原因**：未配置 API Key
**解决**：按照上述步骤配置 `DEEPSEEK_API_KEY`

### 问题：点击后一直加载
**原因**：网络问题或 API Key 无效
**解决**：
1. 检查网络连接
2. 验证 API Key 是否正确
3. 查看浏览器控制台错误

### 问题：分析结果不准确
**原因**：数据不足或质量问题
**解决**：
1. 确保设备正常工作
2. 检查数据完整性
3. 多次分析对比
